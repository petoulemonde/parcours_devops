## Commandes

1. Naviguer dans ce dossier

2. Construire l'image : `podman build . -t first_app:vX` (X = version, pour que je puisses suivre les tailles au fur et à mesure)
Vérifier que l'image a bien été créée : `podman images` (l'image nommée *premier_dockerfile* devrait apparaitre)

Lorsqu'on recrée la même image : `podman build --no-cache . -t first_app:vX` -> empeche podman à réutiliser son cache, et oblige à repartir des fichiers.

3. Run l'image : `podman run -d -p 8000:8000 first_app:vX`
- `-p 8000:8000` : mapper le port 8000 du système avec le port 8000 du container

4. Tester le container : 
```bash
curl http://0.0.0.0:8000/health
curl http://0.0.0.0:8000/hello
curl http://0.0.0.0:8000/text/my_test
```

5. Amélioration de la taille de l'image : 
- `dockerfile_v1` : fichier initiale -> 695 MB
- `dockerfile_v2` : regroupement des RUN -> 246 MB
- `dockerfile` : passage sur alpine -> 120 MB

6. Pour supprimer le conteneur arreté : 
- `podman ps -a` : identifier le nom du conteneur (*PORTS* = *localhost/premier_container*) -> regarder la colonne *NAMES* 
- `podman rm <nom>` : supprime le container arreté
